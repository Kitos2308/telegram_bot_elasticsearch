# telegram_bot

Телеграм бот для поиска critical & error из elasticsearch и информирования  о ошибках в телеграм группы, так же 
проводит аналитику перехода по запрещенным комбинациям, которые можно добавить командой из группы телеграм /items -> 
запрещенные апи -> добавить путь. Идет аналитика разных ip работающих в одной сессии для добавления в block list.

Установка pip3 install -r requirements.txt.
Создать файл config.py в папке проекта и прописать chat_id для телеграм и завести телеграм бота через botfather. 
Обратите внимание, на название индексов в elasticsearch, для микросервисов production на конце названия индекса 
необходимо название _prod, для дев сегмента просто название микросервиса.

launch - python3 app_es_test.py
